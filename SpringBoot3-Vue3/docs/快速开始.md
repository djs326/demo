## 环境准备

### 必需软件

- **JDK 17+** - 后端运行环境
- **Maven 3.8+** - 后端构建工具
- **Node.js 18+** - 前端运行环境
- **npm 9+** 或 **yarn 1.22+** - 前端包管理工具
- **MySQL 8.0+** - 数据库
- **Redis 7.0+** - 缓存

### 可选软件

- **Docker 20.10+** - 容器化部署
- **Docker Compose 2.0+** - 容器编排

## 后端启动

### 1. 克隆项目

```bash
git clone <repository-url>
cd SpringBoot3-Vue3/back
```

### 2. 数据库配置

#### 创建数据库

```sql
CREATE DATABASE springboot_vue3 DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

#### 导入初始化脚本

```bash
mysql -u root -p springboot_vue3 < ../sql/mysql.sql
```

### 3. 修改配置文件

编辑 `back/common/src/main/resources/application.yml`，修改数据库和 Redis 连接信息：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/springboot_vue3?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
    username: root
    password: your_password
  data:
    redis:
      host: localhost
      port: 6379
      password: your_redis_password  # 如无密码可删除
```

### 4. 编译项目

```bash
mvn clean install -DskipTests
```

### 5. 启动后端服务

```bash
cd common
mvn spring-boot:run
```

后端服务将在 `http://localhost:8080` 启动。

## 前端启动

### 管理前端

#### 1. 进入管理前端目录

```bash
cd ../../front-admin
```

#### 2. 安装依赖

```bash
npm install
# 或使用 yarn
yarn install
```

#### 3. 启动开发服务器

```bash
npm run dev
# 或使用 yarn
yarn dev
```

管理前端将在 `http://localhost:5173` 启动。

### 用户前端

#### 1. 进入用户前端目录

```bash
cd ../front-user
```

#### 2. 安装依赖

```bash
npm install
# 或使用 yarn
yarn install
```

#### 3. 启动开发服务器

```bash
npm run dev
# 或使用 yarn
yarn dev
```

用户前端将在 `http://localhost:5174` 启动（或另一个可用端口）。

```bash
cd ../back/common
mvn test
```

## 使用 Docker 快速启动（推荐）

如果您已安装 Docker，可直接使用 Docker Compose 快速启动整个项目：

```bash
cd ..  # 回到项目根目录
docker-compose up -d
```

启动后访问：
- 后端: `http://localhost:8080`
- 管理前端: `http://localhost:8081`
- 用户前端: `http://localhost:8082`
- API 文档: `http://localhost:8080/doc.html`
